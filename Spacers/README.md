# Spacer - 5mm

## Overview

Provides a 5mm spacer for the OpenAero design system, intended to raise the height of any accessory.

### Specs

Height: 5mm\
Width: 21mm\
Length: 47mm\

### Accessory Mount Spec

Base Mount: 21mm x 47mm (Standard)\
Bolt Length Required: 5mm\

### Hardware Required
None

### Installation Notes
None
